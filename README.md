# ipget
A cookies-less, spy-less way to get your IPv4 and IPv6 addresses using Ipify.org API.

## Usage

Go on the web page at [dtztim.github.io/ipget].

This tool will send a request to ipify.org API for IPv4 <br>
and IPv6 addresses in one go so no need to wait. 

## Attributions

This tool is entierly using [Ipify.org API](https://ipify.org) to get the user IPs.
